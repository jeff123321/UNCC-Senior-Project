Some important things to mention:

You may remember from math class that parentheses can be used to signify multiplication. Like 5(3) = 15. You can't do that in C++, you will get an error. Instead type 5 * 3.

Use parentheses if you want to influence the order the operations will be evaluated in.

5 * 4 + 2 = 22

5 * (4 + 2) = 30

The modulus operator will return the remainder after you perform division with two integers.

5 % 2 = 1.

2 goes into 5 twice, and the remainder is 1. The modulus operator is a very useful tool, and we will see why it is so important in future exercises.

One last thing to mention: The equals sign is for assigning new values to your variables. You can even use this to update the value of your number based off its previous value:

int num = 5;
num = num + 1;
//now the number is 6

num = num + 1 may seem like a mathematical absurdity, but remember that the equal sign is used for assigning values, not necessarily what it means in math class.

Another way to write this is:

num += 1;

This does the same thing as above, but now there is no mathematical absurdity.

Now, let us dive into some exercises.

Exercise 1: Area of a rectangle

Write a program that will ask the user for the length and width of a rectangle. The program will then multiply the two numbers and show the area of the rectangle.

Exercise 2: Area of a circle

Write a program that will ask the user for the radius of a circle. Then, show the user the area of the circle by doing radius times radius times PI.

*make sure you make the value PI a constant.

const float PI = 3.14;

Exercise 3: Swapping two values

Write a program that will ask the user for two integers, the program will store these as num1 and num2. Then the program will swap the values of num1 and num2, and then print these to the screen.

Hint: If you just do

num1 = num2;
num2 = num1;

Then both numbers will be the same. You need to create a third variable, num3, to help in the switching.

Exercise 4: Finding the average

Write a program that will ask the user for three numbers. These numbers will be added up and divided by three, which will give you the average. Use your parentheses wisely, and be careful with your division. We want our answer to allow for decimals.

Exercise 5: Converting temperatures

Write a program that will ask the user for a temperature in Celsius. Then convert that temperature to Fahrenheit. If you don't remember, the formula is multiply the temperature by 1.8, and then add 32.

Exercise 6: Total cost after tax

Write a program that will ask the user for the cost of an item. Then figure out the total cost after sales tax is added. Assume a tax rate of 7.5%. Store the tax rate as a constant.

Hint: You will be multiplying the cost by (1 + 0.075)

Whew, that's a lot of exercises. If you could make it through all that, then you are in good shape to move on to the next section!
