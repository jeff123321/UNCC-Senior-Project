Pointers

Pointers are a pretty cool feature of C++. Other languages try to hide pointers from you, but C++ is all about them.

A pointer is a special kind of variable that instead of storing a value like 5 or "Johnny" stores the address of another variable.

Here is an example:

This is how I can create the pointer.

int * addressOfNum;

The int* means that the pointer will hold the address of an integer variable.

To put an address in my pointer, I can do this:

int * addressOfNum = #

The ampersand means that I am referring to the address of the variable called num. This is how I set the pointer to point to the address of the variable num.

If you notice, when I printed the pointer to the screen we got something that looked like 0x followed by letters and numbers. This is how addresses look like in computers. The letters and numbers are actually in hexadecimal, which is kind of number system based around the number 16.

To print the value pointed to by the pointer, we use the dereference operator:

*addressOfNum;

The way I think of it is like that the asterisk is basically telling the pointer to spill its guts out. So instead of getting a hexadecimal address, we get the value of num.

Arrays are very closely related to the idea of pointers, since elements in an array are supposed to be right next to each other in memory.

There are some cool things you can do with pointers. Sometimes it is faster to work with addresses of information rather than the data itself.

Structures

Structures are used to group data under one label. Structures are created outside of your main. Here is an example so you can see the structure:

struct cat
{
      string name;
      float weight;
      string color;
      int age;
};

Cat is the name of the structure. Name, weight, color, and age are the members of the structure.

Here is a program the puts the structure to use:

First, I had to create an instance of the cat structure.

cat myCat;

This works a lot like how we have been creating variables up to now, where we say int num or whatever. Except instead of creating an integer named num, we are creating a cat data structure named myCat.

myCat.name = "Paul";
myCat.weight = 20;
myCat.color = "orange";
myCat.age = 15;

You can access the member variables of your structure by typing the name of the structure you created, followed by a dot, and then the name of the member variable that you want to access. These member variables can be treated just like any of the other variables you have been creating up until this point.

There are other ways to create structures.

struct cat
{
      float weight;
      string color;
      int age;
}Paul, Tom, Penelope;

This is similar to what we have above, except this time the names of our cats have been included next to the structure definition. This means we do not have to create instances of our cat data structure, they already created. Here is a program using this:

Classes

Now we are starting to get more into object oriented type stuff, which is beyond what I had originally planned for with this site. However, a light introduction should be okay. Just keep in mind that there is a lot more to learn about with this stuff than what I am going to put here.

Classes are like structures, except instead of just holding data they can also have functions.

Here is an example of a class circle:

class circle
{
      float radius;

      public:
      void setRadius(float r)
      {
            radius = r;
      }
      float getRadius()
      {
            return radius;
      }
      float circumference()
      {
            return 2 * radius * PI;
      }
      float area()
      {
            return radius * radius * PI;
      }
      float diameter()
      {
            return radius * 2;
      }
};

The class circle has a member variable named radius for holding the radius of a circle, but there are also functions for calculating the diameter, the circumference, and the area. The public: before the functions just makes it so that we can access these functions outside of the class.

Here is a program that puts our circle class to use:

This is just like creating integers named num1 and num2, except here we are creating circle objects and calling them myCircle and yourCircle.

The rest of the main is us calling the various functions belonging to our circle objects.

myCircle.setRadius(10)
myCircle.getRadius()
myCircle.diameter()
myCircle.circumference()
myCircle.area()

These are accessed by putting the name of the object, a dot, and the name of the function we want to use.
