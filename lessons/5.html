Lesson 5: Decision Making

We want our programs to be able to perform different actions depending on certain conditions. To do this, we will use if..else... statements. These are used in all kinds of programming languages, not just C++. Here is the basic structure of an if.. else.. statement:

if(some condition)
{
      some things to do
}
else
{
      if the condition is not met, do this instead
}

Note the else part is optional. Also, if there is just one line of code after your if, then the curly braces are optional. If you want to check for multiple conditions, you can use if..else if.. else structure.

if(some condition)
{
      some things to do
}
else if(another condition)
{
      some other things to do
}
else
{
      if all else fails, do this
}

You can put as many else ifs as you want.

Cases

Let us look at another kind of decision making tool: case structures.

Case structures are useful when a variable can take one of several different values, and you want to have something planned for each of them. Here is the basic structure of one.

switch(num)
{
      case 1:
            something to do
      break;
case 2:
      something to do
            break;
case 3:
            something to do
            break;
default:
            something to do
            break;
}

The default will execute if the number does not meet any of the above criteria. The breaks that you see are due to a peculiarity in how case statements work in C++. Without the breaks, all the code below the case will get executed.

Exercise 1: Which one is larger?

The user will enter in two numbers. The program will tell the user which number they entered is larger.

Exercise 2: Even or odd

The user will enter an integer. Tell them if it is even or odd. Use the modulus operator (%).

Exercise 3: What is the letter grade?

The user will enter the percentage they got on the test. Tell them what letter grade they deserve. Use the 7 point scale:

100 - 93: A
92 - 85: B
84 - 77: C
77 - 70: D
69 and below: F

Exercise 4: F to C, and vice versa.

The user will enter in a temperature. Then the program will ask you if that temperature is in Fahrenheit or Celsius. The user will enter either an F or a C. Give them the temperature in the other scale.

F = (5/9) * C + 32
C = (F - 32) * (9/5)

You can easily figure out these formulas if you can remember the boiling points in these two scales.
