Lesson 2: "Hello World"

Programmers are sticklers for tradition. So, when introducing a new programming language, it is customary to begin by showing a little program which simply prints "Hello world" to the screen.

Have a looksee at the code below you. Click the run button next to the little panel below to see what it does.

Try experimenting with changing things around, see what happens. You might get a few errors, but that's okay. Learn what works and what doesn't. Particularly see if you can print other things to the screen besides "hello world".

Now is the time I give you some guidelines on basic formatting in C++. One thing that you will very quickly find out is that C++ is very picky about making sure that everything is formatted correctly. Simply misplacing a piece of punctuation might make it so that the program will not work. Here are some rules:

1. Every time you use an opening brace { there must also be a closing brace }.

2. The same rule applies to parentheses. ( )

3. Almost every statement you write must have a semicolon at the end. This is one of the weird things about C++, since not all languages require you to do this. You will notice from the program above that "#include <iostream>" and "int main()" do not end with semicolons. The reason for that is simple: they are just exceptions.

4. Anything you write after two forward slashes will be ignored by the compiler. Here is an example:

//ergerjhfigheiogheioghiohgiohg

5. When writing cout statements, you just have to follow the format.

cout << "Your text here";

You can put comments in your code and it will not affect the output. The two slashes indicate that you are about to write a comment. Comments are used to provide explanations for what the code does and how it works. These are very important especially when other people need to read your code. Here it is in action.

If you want to move to the next line after printing something, try this:

cout << "This is the first line." << endl;

cout <<"This is the second line.";

The endl is short for end line, hopefully that is not too confusing. Without it, both of those statements would show up on the same line. Another way to move to the next line is with the newline character '\n'.

cout << "This is the first line.\n";

cout <<"This is the second line.";

This requires less typing which you may prefer.

We will go into making your output prettier in later lessons.
