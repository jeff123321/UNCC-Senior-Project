Lesson 7: Arrays

An array is used to hold a bunch of variables so that they will all be next to each other in memory, so that they can all be easily accessed. Think of it like having a shelf with a bunch of books on it. Here is how to create and array that will hold 5 integers:

int[5] arr;

The value inside the square brackets is the size of the array.

This is how you access items in your array:

arr[0];
arr[1];
arr[2];
arr[3];
arr[4];

The address of the first item is 0. The address of the last item is going to be the size of the array minus 1. So arr[4] is the last item in the array.

Each element in the array can be treated as its own variable. You can perform math on it, you can set its value. Anything that you can do with a variable you can do with an array item. The code below plays around with elements in the array:

You can populate the contents of an array when you create it, like so:

int arr[] = {1, 2, 3, 4, 5 };

You can leave out the size of the array when you are initializing it. The compiler is smart enough to know that if you initialize an array with 5 values then the size of the array must also be 5. However, if you do not initialize, then you will need to specify the size.

When you create an array without initializing it, the array will be full of junk data.

However, if you initialize one of the values, then all of the other ones will be initialized to 0.

Of course, you can create arrays for all the variable types: floats, doubles, chars, and strings. Actually, strings themselves are just arrays of characters. The original C programming language didn't even have strings, just character arrays.

We talked about for loops in the previous section. For loops are particularly useful for accessing the elements of array. The program below fills each item with an array with random numbers.

I haven't gone into random numbers yet, so I thought I would bring it up here. The procedure for creating arrays is as follows:

You will need to put these at the top of your code:

#include <cstdlib>
#include <ctime>

Next you will need to seed the random number generator. The easiest way to do this is to use this line of code:

srand(time(NULL));

This uses the current time as the seed for the random number generator.

Then whenever you want a random number, you can do something like this:

int randomNumber = rand();

If you want your random number to be in a certain range, you can use the modulo operator:

int randomNumber = rand() % 6;

This will give you a number between 0 and 5. If you want the number to be between 1 and 6, simply add one to the random number:

int randomNumber = rand() % 6 + 1;

This will give you a random number between 1 and 6, which would be good to simulate a dice roll. If you are interested in completing the casino project (listed in the projects tab), then you can refer back to this.

One last thing I want to mention about arrays. If you want to copy an array, you might think it would be as simple as this:

yourArray = myArray;

But in actuality, this will not work. To copy the contents of an array to another: you will need to use a for loop like so:

for(int i = 0; i < 10; i++)
{
      yourArray[i] = myArray[i];
}
